# Image Classification using Teachable Machine

## Project Description
This project uses Google's Teachable Machine to classify images using a TensorFlow/Keras model.

## Files
- Untitled0.ipynb
- keras_model.h5
- labels.txt
- screenshot.png

## How to Run
1. Open the notebook in Google Colab.
2. Upload the model files and test image.
3. Run all cells.
4. The model predicts the image class and confidence score.
